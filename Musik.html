Controller für Windows Lautstärkemixer:

Eine kleine Bastelei, die es einem ermöglicht, den Windows Lautstärkemixer über Drehregler zu Bedienen.
Ein Arduino ließt einen Spannungsteiler (Potentiometer) aus, das Signal wird an den Rechner übertragen und über die Software interpretiert.
Es können jedem Regler eines oder mehrer Programme zugewiesen werden.
Testweise habe ich mir so ein tolles Ding mal gebaut, in zukunsft soll eine etwas schönere und funktionale Version folgen
Die kosten lagen unter 15€, kommerziell Lösungen liegen bei mind. 80€ und sind deutlich eingeschränkter durch ihre Software.
